## Quick Usage

```html
<ck-multiselect-grid
  title="Select Resource Scopes"
  description="Choose which resource scopes this client is allowed to request."
></ck-multiselect-grid>
```

### Attributes

- `title`: label text
- `description`: helper text (preferred)
- `discription`: helper text (legacy spelling)
