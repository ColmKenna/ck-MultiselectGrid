Responsive Design

Applies different class names to the internal fieldset via fieldset-class.

@media (max-width: 600px) {
  .responsive fieldset.multiselect-fieldset {
    padding: 0;
  }
}
@media (min-width: 1024px) {
  .responsive fieldset.multiselect-fieldset {
    padding: 24px;
  }
}
